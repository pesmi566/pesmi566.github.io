📘 Chapitre 6 : Fonctions affines et équations de droites

6.1 Différence : fonction linéaire vs fonction affine

📖 Les deux types

Fonction LINÉAIRE :

f(x) = ax   (pas de terme constant !)

La droite passe par l'origine (0 ; 0)

Représente une situation de proportionnalité

Exemple : f(x) = 0,5x

Fonction AFFINE :

f(x) = ax + b

La droite ne passe généralement pas par l'origine

La fonction linéaire est un cas particulier (b = 0)

Exemple : g(x) = 0,2x + 6

✏️ Exemple de la vie réelle (du cours)

Alberto : 0,50 € / km → y = 0,5x → Fonction linéaire (pas de forfait fixe)

Bernardo : 6 € de forfait + 0,20 € / km → y = 0,2x + 6 → Fonction affine

💡 Bernardo facture 6 € même si on ne roule pas du tout → c'est le "b".

Les deux droites se croisent pour x = 20 km : pour 20 km, les frais sont identiques (10 €).

6.2 Définition officielle et cas particuliers

📐 Définition officielle

Une fonction f définie par f(x) = ax + b (a, b ∈ ℝ) est appelée une fonction affine.

Dans l'équation y = ax + b d'une droite d :

  • a est appelé le coefficient directeur de la droite d
  • b est appelé l'ordonnée à l'origine de la droite d
    (la droite passe par le point de coordonnées (0 ; b))

Tableau des cas particuliers :

NomForme f(x)Valeur de aValeur de bExemple
Fonction affineax + bquelconque ≠ 0quelconquef(x) = 2x – 5
Fonction linéaireaxquelconque ≠ 0b = 0f(x) = 3x
Fonction constanteba = 0quelconquef(x) = –4
⚠️ Fonctions qui NE SONT PAS affines — à connaître !
  • f(x) = 4x² → ❌ Fonction carré (degré 2)
  • f(x) = 3/(5x) → ❌ Fonction inverse (x est au dénominateur)
  • f(x) = –3 – 5x² → ❌ Degré 2
  • f(x) = √5 · x – 5 → ✅ AFFINE ! a = √5 est un réel, b = –5

6.3 Identifier une fonction affine (mettre sous la forme ax + b)

🔧 Le truc fondamental

On développe et simplifie l'expression, puis on identifie :

  • a = le coefficient devant x (le nombre qui multiplie x)
  • b = le terme sans x (le nombre "seul")

Si on peut écrire f(x) = ax + b → c'est une fonction affine.

✏️ Exemples complets (Exercices 1 & 2 du cours)

1) f(x) = 5x + 1 → déjà sous la forme → Affine, a = 5, b = 1

2) f(x) = 4x² → degré 2 → PAS affine (fonction carré)

3) f(x) = –5x → f(x) = –5x + 0 → Linéaire, a = –5, b = 0

4) f(x) = –4 → f(x) = 0·x + (–4) → Constante, a = 0, b = –4

5) f(x) = 7 – 3x → f(x) = –3x + 7 → Affine, a = –3, b = 7

6) f(x) = 2(x – 1) – 4(x + 3)
= 2x – 2 – 4x – 12 = –2x – 14 → Affine, a = –2, b = –14

7) f(x) = (x + 1)² – x²
= x² + 2x + 1 – x² = 2x + 1 → Affine, a = 2, b = 1

8) f(x) = x² – (x – 2)(x – 2)
= x² – (x² – 4x + 4) = x² – x² + 4x – 4 = 4x – 4 → Affine, a = 4, b = –4

9) f(x) = (3x – 2)/3
= 3x/3 – 2/3 = x – 2/3 → Affine, a = 1, b = –2/3

6.4 Représentation graphique d'une fonction affine

📖 Propriété centrale

La représentation graphique d'une fonction affine f(x) = ax + b est une droite d d'équation y = ax + b.

  • Si a > 0 : la droite monte ↗
  • Si a < 0 : la droite descend ↘
  • Si a = 0 : droite horizontale →

Méthode 1 : Tableau de valeurs (toujours possible)

🔧 Étapes
  1. Choisir au moins 2 valeurs de x (ex : 0 et 1, ou 0 et 2)
  2. Calculer f(x) pour chaque valeur
  3. Placer les points dans le repère
  4. Tracer la droite passant par ces points (prolonger dans les deux sens)
✏️ Exemple : f(x) = 2x + 5

Si x = 0 : f(0) = 2·0 + 5 = 5 → Point A(0 ; 5)

Si x = –3 : f(–3) = 2·(–3) + 5 = –6 + 5 = –1 → Point B(–3 ; –1)

On trace la droite passant par A et B.

💡 b = 5 = ordonnée à l'origine → le point (0 ; 5) est là où la droite coupe l'axe y.

Méthode 2 : b + coefficient directeur (Δy/Δx)

🔧 Utiliser le taux d'accroissement
  1. Placer l'ordonnée à l'origine : le point (0 ; b) sur l'axe y
  2. Utiliser a = Δy/Δx pour trouver un 2ème point
    Ex : si a = 2/3, aller 3 cases à droite (Δx=3) et 2 cases en haut (Δy=2)
    Ex : si a = –2/3, aller 3 cases à droite et 2 cases en bas
  3. Tracer la droite passant par ces deux points

Un point appartient-il à une droite ? — Méthode clé de la Prüfung

📐 Propriété fondamentale (Exercice 2 de la Prüfung)

Un point D(x₀ ; y₀) appartient à la droite d ⟺ f(x₀) = y₀

→ On substitue la coordonnée x dans f(x) et on vérifie si on obtient la coordonnée y.

✏️ Exemple complet — Prüfung Exercice 2 : f(x) = –4x + 1
Question : A(1 ; –3) appartient-il à la droite ?

f(1) = –4·1 + 1 = –4 + 1 = –3
Comme f(1) = –3 = y_A ✅ → A(1 ; –3) appartient à la droite.

Question : B(2/3 ; –2) appartient-il à la droite ?

f(2/3) = –4·(2/3) + 1 = –8/3 + 3/3 = –5/3
Comme f(2/3) = –5/3 ≠ –2 = y_B ❌ → B(2/3 ; –2) n'appartient pas à la droite.
⚠️ Erreur fréquente — Prüfung

Pour vérifier si B(2/3 ; –2) appartient à une droite :

❌ FAUX : calculer f(–2) → c'est la deuxième coordonnée, pas la première !

✅ JUSTE : toujours substituer la coordonnée x (la première) dans f(x)

La coordonnée x est toujours la première dans (x ; y).

6.5 Coefficient directeur – taux d'accroissement

a = Δy / Δx = (f(x₂) – f(x₁)) / (x₂ – x₁)
📐 Formule essentielle

Pour deux points x₁ et x₂ distincts d'une droite :

a = (accroissement des ordonnées) / (accroissement des abscisses) = Δy / Δx

Cette formule permet de calculer le coefficient directeur à partir de n'importe quels deux points de la droite.

✏️ Exemples du cours

Ex 1 : f(0) = 2 et f(1) = –3. Calculer a :

a = (f(1) – f(0)) / (1 – 0) = (–3 – 2) / 1 = –5
Donc f(x) = –5x + 2

Ex 2 : f(–3) = 1 et f(2) = 5. Calculer a :

a = (f(2) – f(–3)) / (2 – (–3)) = (5 – 1) / (2 + 3) = 4/5 = 4/5
Valeur de aLa droite…
a > 0monte de gauche à droite ↗ (croissante)
a < 0descend de gauche à droite ↘ (décroissante)
a = 0est horizontale (fonction constante) →
|a| grandest très inclinée (pente raide)
|a| petitest presque horizontale (pente douce)

6.6 Trouver l'expression algébrique à partir de deux points/valeurs

🔧 Méthode en 4 étapes — LA méthode la plus importante !
  1. Poser f(x) = ax + b (car f est affine)
  2. Calculer a avec la formule a = (y₂ – y₁)/(x₂ – x₁)
    Astuce : Si on connaît f(0), alors b = f(0) directement !
  3. Trouver b en substituant les coordonnées d'un point dans f(x) = ax + b
  4. Écrire la réponse : f(x) = ax + b
✏️ Exemple 1 — Prüfung Exercice 4.1 : f(0) = 3 et f(2) = –5
Étape 1 : f est affine → f(x) = ax + b

Étape 2 : f(0) = 3 → b = f(0) = 3  (astuce directe !)

Étape 3 : f(2) = –5 → a·2 + 3 = –5
              ⟺ 2a = –5 – 3 = –8
              ⟺ a = –4

Résultat : f(x) = –4x + 3
✏️ Exemple 2 — Prüfung Exercice 4.2 : g(2) = 3 et g(–1) = 4
Étape 1 : g est affine → g(x) = ax + b

Étape 2 : a = (g(–1) – g(2)) / (–1 – 2) = (4 – 3) / (–3) = 1/(–3) = –1/3

Étape 3 : g(x) = –(1/3)x + b
           g(2) = 3  →  –(1/3)·2 + b = 3
                  →  –2/3 + b = 3
                  →  b = 3 + 2/3 = 9/3 + 2/3 = 11/3

Résultat : g(x) = –(1/3)x + 11/3
✏️ Exemple 3 — Exercice 8 du cours : f(0) = 2 et f(5) = 4
Astuce : f(0) = 2 → b = 2 directement !

Calcul : a = (f(5) – f(0)) / (5 – 0) = (4 – 2) / 5 = 2/5

Résultat : f(x) = (2/5)x + 2
💡 Astuce clé — À mémoriser !

Si on connaît f(0), alors b = f(0) directement ! Pas besoin de calcul supplémentaire.

Exemple : f(0) = –7 → b = –7 immédiatement.

Trouver x tel que f(x) = k

✏️ Exemple — Prüfung Exercice 2b : f(x) = –4x + 1, trouver x tel que f(x) = –4
f(x) = –4  ⟺  –4x + 1 = –4   | –1
           ⟺  –4x = –5        | :(–4)
           ⟺  x = 5/4

6.7 Équations de droites

📐 Rappel — Les 3 types d'équations
ÉquationType de droiteC'est une fonction ?
y = ax + bDroite oblique ou horizontale✅ OUI
y = c (constante)Droite horizontale (parallèle à l'axe x)✅ OUI — fonction constante
x = c (constante)Droite verticale (parallèle à l'axe y)❌ NON — pas une fonction !
⚠️ Important — Droite verticale ≠ fonction !

Une droite d'équation x = c (ex : x = 2, x = –4) n'est PAS une fonction.

Pour x = 2, il y aurait une infinité de valeurs y → impossible d'associer une seule image.

Mais on peut quand même lui donner une équation et la dessiner dans un repère.

Trouver l'équation d'une droite — Méthode complète

🔧 Méthode — 3 cas différents

Cas 1 : On connaît le coefficient directeur a et un point (x₀ ; y₀)

  1. Écrire y = ax + b avec le a donné
  2. Substituer (x₀ ; y₀) pour trouver b : y₀ = a·x₀ + b → b = y₀ – a·x₀
  3. Écrire la réponse : y = ax + b

Cas 2 : On connaît deux points (x₁;y₁) et (x₂;y₂)

  1. Calculer a = (y₂ – y₁)/(x₂ – x₁)
  2. Substituer un des deux points pour trouver b
  3. Écrire la réponse

Cas 3 : Droite parallèle à un axe

  • Parallèle à l'axe des abscisses (axe x) → y = k (k = l'ordonnée du point)
  • Parallèle à l'axe des ordonnées (axe y) → x = k (k = l'abscisse du point)
✏️ Exercice 5 de la Prüfung — 3 sous-questions

5.1 : a = 2, point A(5 ; –5)

y = 2x + b
A(5 ; –5) ∈ d₁  →  –5 = 2·5 + b  →  –5 = 10 + b  →  b = –15
d₁ : y = 2x – 15

⚠️ ERREUR FRÉQUENTE : confondre a (= 2, donné) avec la coordonnée y du point (= –5) !
   Toujours utiliser le a de l'énoncé, pas les coordonnées du point.

5.2 : Points B(2 ; –3) et C(–10 ; 5)

a = (5 – (–3)) / (–10 – 2) = 8 / (–12) = –2/3

y = –(2/3)x + b
B(2 ; –3) ∈ d₂  →  –3 = –(2/3)·2 + b
                 →  –3 = –4/3 + b
                 →  b = –3 + 4/3 = –9/3 + 4/3 = –5/3
d₂ : y = –(2/3)x – 5/3

5.3 : Point C(2 ; –12), parallèle à l'axe des abscisses

Parallèle à l'axe des abscisses = droite horizontale
Équation de la forme y = k → k = l'ordonnée du point = –12
d₃ : y = –12

Lire les équations sur un graphique (Prüfung Exercice 7)

🔧 Méthode — Identifier visuellement
  1. Trouver l'ordonnée à l'origine b : où la droite coupe l'axe y
  2. Choisir 2 points "propres" sur la droite (à des coordonnées entières)
  3. Calculer a = Δy/Δx avec ces deux points
  4. Écrire l'équation y = ax + b

Cas spéciaux :

  • Droite horizontale → y = b, a = 0
  • Droite verticale → x = c (pas une fonction !)
  • Droite passant par l'origine → b = 0, donc y = ax

6.8 Droites parallèles

d ∥ d' ⟺ a_d = a_d'
📐 Propriété des droites parallèles

Deux droites (non verticales) sont parallèles si et seulement si elles ont le même coefficient directeur.

C'est-à-dire : même pente, mais ordonnées à l'origine différentes.

✏️ Exemples

Parallèles ✅ : d : y = –3x – 2 et d' : y = –3x + 4 → a_d = –3 = a_d' ✅

PAS parallèles ❌ : d₁ : y = (3/2)x + 5 et d₂ : y = –(3/2)x + 5

→ a₁ = 3/2 ≠ –3/2 = a₂ → PAS parallèles !

💡 Même valeur absolue MAIS signes opposés → coefficients directeurs différents → PAS parallèles !

C'est l'erreur de la Prüfung Exercice 3.2 ! Les élèves ont confondu |3/2| = |–3/2|.

Trouver une droite parallèle passant par un point donné

🔧 Méthode
  1. La nouvelle droite a le même a que la droite de référence
  2. Écrire y = ax + b avec ce a
  3. Substituer le point donné pour calculer b
  4. Écrire la réponse
✏️ Exemple (Exercice 28 du cours)

Droite d passant par A(–4 ; –2), parallèle à d₂ : 2x + y – 3 = 0

D'abord réécrire d₂ : y = –2x + 3  →  a₂ = –2
Donc d : y = –2x + b
A(–4 ; –2) ∈ d  →  –2 = –2·(–4) + b  →  –2 = 8 + b  →  b = –10
d : y = –2x – 10

📗 Chapitre 7 : Systèmes de deux équations à deux inconnues

7.1 Définition et résolution graphique

📖 Définition

Un système de deux équations du premier degré à deux inconnues x et y est de la forme :

{ ax + by + c = 0
{ a'x + b'y + c' = 0

Résoudre un système = trouver le(s) couple(s) (x ; y) qui vérifient les deux équations simultanément.

La solution s'écrit S = {(x ; y)} — avec des parenthèses pour le couple !

Situation géométriqueNombre de solutions
Les deux droites se croisent en un pointUne unique solution
Les deux droites sont parallèles (même a, b différent)Aucune solution
Les deux droites sont confondues (identiques)Infinité de solutions

Résolution graphique

🔧 Méthode graphique
  1. Réécrire chaque équation sous la forme y = ax + b
  2. Tracer les deux droites dans le même repère orthogonal
  3. Lire les coordonnées du point d'intersection → c'est la solution

⚠️ La résolution graphique est approximative. Pour la valeur exacte → calcul algébrique.

✏️ Exemple — Prüfung Exercice 8.1 : {4x – 2y = 6 ; 2x + y = 4}
Éq. 1 : 4x – 2y = 6  →  2y = 4x – 6  →  y = 2x – 3
Éq. 2 : 2x + y = 4   →  y = –2x + 4

Tracer les deux droites → point d'intersection ≈ (1,75 ; 0,5)
S = {(7/4 ; 1/2)}  [valeur exacte par calcul]

7.2 Méthode par substitution

📖 Principe

On exprime une inconnue en fonction de l'autre dans une équation, puis on substitue cette expression dans la deuxième équation. On obtient une équation à une seule inconnue.

🔧 Méthode par substitution — 7 étapes
  1. Choisir l'équation la plus simple (idéalement celle avec un coefficient = 1)
  2. Exprimer une inconnue en fonction de l'autre dans cette équation
    Ex : de x + 4y = 3, tirer x = 3 – 4y
  3. Appeler cette expression (3) pour s'y référer
  4. Substituer (3) dans l'AUTRE équation — JAMAIS dans la même !
  5. Résoudre l'équation à une inconnue obtenue
  6. Substituer la valeur trouvée dans (3) pour calculer la 2ème inconnue
  7. Écrire S = {(x ; y)} et vérifier dans les deux équations
✏️ Exemple complet du cours : {2x + 3y = –4 ; 4x – 5y = –30}
{ 2x + 3y = –4    ...(1)
{ 4x – 5y = –30   ...(2)

Étape 1 : Choisir (1) et exprimer x :
(1)  ⟺  2x = –3y – 4
     ⟺  x = –(3/2)y – 2    ...(3)

Étape 2 : Substituer (3) dans (2)  [PAS dans (1) !]
4·(–(3/2)y – 2) – 5y = –30
⟺ –6y – 8 – 5y = –30
⟺ –11y – 8 = –30
⟺ –11y = –22
⟺ y = 2    ...(4)

Étape 3 : Substituer (4) dans (3) :
x = –(3/2)·2 – 2 = –3 – 2 = –5

Résultat : S = {(–5 ; 2)}

Vérification :
(1) : 2·(–5) + 3·2 = –10 + 6 = –4 ✅
(2) : 4·(–5) – 5·2 = –20 – 10 = –30 ✅
✏️ Exemple simple : {y = 2x ; 2x – 3y = 4}
y = 2x est déjà exprimé ! → substituer directement dans la 2ème :
2x – 3·(2x) = 4
⟺ 2x – 6x = 4
⟺ –4x = 4  →  x = –1
y = 2·(–1) = –2
S = {(–1 ; –2)}
✏️ Exemple — Prüfung Exercice 8.2b : {y = –5 + 3x ; x + 3y = 10}
{ y = –5 + 3x    ...(1)
{ x + 3y = 10    ...(2)

(1) donne y directement → substituer dans (2) :
x + 3·(–5 + 3x) = 10
⟺ x – 15 + 9x = 10
⟺ 10x = 25  →  x = 5/2

Dans (1) : y = –5 + 3·(5/2) = –5 + 15/2 = –10/2 + 15/2 = 5/2

S = {(5/2 ; 5/2)}

7.3 Méthode par combinaisons linéaires

📖 Principe

On multiplie les équations par des nombres pour que les coefficients d'une inconnue soient opposés (l'un est +k, l'autre –k). En additionnant les deux équations membre à membre, cette inconnue disparaît.

🔧 Méthode par combinaisons linéaires — 6 étapes
  1. Choisir quelle inconnue éliminer (ex : éliminer x)
  2. Multiplier une ou les deux équations par des nombres appropriés
    But : avoir des coefficients opposés pour cette inconnue
    Ex : coefficients 2x et 4x → multiplier la 1ère par (–2) pour avoir –4x et +4x
  3. Additionner les deux équations membre à membre → l'inconnue choisie disparaît
  4. Résoudre l'équation à une inconnue obtenue
  5. Substituer cette valeur dans une des équations originales pour trouver la 2ème inconnue
  6. Écrire S = {(x ; y)} et vérifier
✏️ Exemple du cours : {2x + 3y = –4 ; 4x – 5y = –30}
{ 2x + 3y = –4    ...(1) × (–2)
{ 4x – 5y = –30   ...(2)

(1) × (–2) :  –4x – 6y = 8    ...(1')
(2) :          4x – 5y = –30   ...(2)

(1') + (2) :  (–4x + 4x) + (–6y – 5y) = 8 + (–30)
              0 – 11y = –22
              y = 2

Dans (1) : 2x + 3·2 = –4  →  2x = –10  →  x = –5
S = {(–5 ; 2)}
✏️ Exemple — Prüfung Exercice 8.2a : {2x – 3y = –9 ; 4x – 3y = –5}
{ 2x – 3y = –9    ...(1) × (–1)
{ 4x – 3y = –5    ...(2)

(1) × (–1) :  –2x + 3y = 9     ...(1')
(2) :          4x – 3y = –5    ...(2)

(1') + (2) :  2x = 4  →  x = 2

Dans (1) : 2·2 – 3y = –9  →  4 – 3y = –9  →  –3y = –13  →  y = 13/3
S = {(2 ; 13/3)}
💡 Quelle méthode choisir ?
SituationMéthode recommandée
Une équation donne déjà y = … ou x = …Substitution (plus rapide !)
Une des équations a un coefficient 1 devant x ou ySubstitution
Les coefficients d'une inconnue sont identiques ou facilement rendus opposésCombinaisons linéaires
Coefficients compliqués / fractionsCombinaisons linéaires
⚠️ Erreurs fréquentes avec les systèmes — À éviter !
  • Substituer dans la MÊME équation (il faut toujours substituer dans l'AUTRE !)
  • Oublier de changer le signe lors de la multiplication par un nombre négatif
    Ex : –(2x – 3y = –9) → –2x + 3y = +9 (les DEUX membres changent de signe !)
  • Calculer une seule inconnue et oublier de calculer la deuxième
  • Écrire S = {–5 ; 2} au lieu de S = {(–5 ; 2)} — le couple a des parenthèses !
  • Ne pas vérifier la solution dans les DEUX équations
  • Confondre les équations du système (travailler avec une équation déjà modifiée)

7.4 Résolution de problèmes en contexte

🔧 Méthode générale en 6 étapes
  1. Lire attentivement et identifier les deux quantités inconnues
  2. Nommer ces quantités : x = … et y = …
  3. Écrire deux équations qui traduisent mathématiquement le problème
  4. Résoudre le système d'équations par substitution ou combinaisons linéaires
  5. Vérifier que la solution correspond aux données du problème
  6. Formuler la réponse en une phrase complète
✏️ Exemple 1 — MathFon Inc. (Activité 1 du cours)

EU : 16 M téléphones + 8 M écouteurs = 2000 M$. USA : 25 M téléphones + 10 M écouteurs = 3000 M$.

Variables : x = bénéfice/téléphone, y = bénéfice/écouteur (en M$)

{ 16x + 8y = 2000   ...(1) × 5
{ 25x + 10y = 3000  ...(2) × (–4)

(1)×5 :     80x + 40y = 10000
(2)×(–4) : –100x – 40y = –12000

Somme : –20x = –2000  →  x = 100

Dans (1) : 16·100 + 8y = 2000  →  1600 + 8y = 2000  →  8y = 400  →  y = 50

Le bénéfice par téléphone est 100 M$ et par écouteur est 50 M$.
✏️ Exemple 2 — Classeur et cahier (Exercice 4 du cours)

Un classeur coûte 1 € de plus qu'un cahier. 2 classeurs + 3 cahiers = 17 €.

Variables : x = prix d'un classeur, y = prix d'un cahier

{ x = y + 1       ...(1)
{ 2x + 3y = 17   ...(2)

Substituer (1) dans (2) :
2(y + 1) + 3y = 17
⟺ 2y + 2 + 3y = 17
⟺ 5y = 15  →  y = 3

Dans (1) : x = 3 + 1 = 4

Un classeur coûte 4 € et un cahier coûte 3 €.
Vérification : 2·4 + 3·3 = 8 + 9 = 17 ✅
✏️ Exemple 3 — Bouquets de fleurs (Exercice 7 du cours : Activité 2)

3 iris + 4 roses = 4,80 €. 5 iris + 6 roses = 7,50 €.

Variables : x = prix d'un iris, y = prix d'une rose

{ 3x + 4y = 4,80   ...(1) × 3
{ 5x + 6y = 7,50   ...(2) × (–2)

(1)×3 :    9x + 12y = 14,40
(2)×(–2) : –10x – 12y = –15,00

Somme : –x = –0,60  →  x = 0,60

Dans (1) : 3·0,60 + 4y = 4,80  →  1,80 + 4y = 4,80  →  4y = 3,00  →  y = 0,75

Un iris coûte 0,60 € et une rose coûte 0,75 €.
